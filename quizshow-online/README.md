# Quizshow Online

## Lokal testen
1. Node.js installieren
2. Ordner öffnen
3. npm install
4. npm start
5. Browser öffnen: http://localhost:3000

## Auf Render hosten
1. GitHub-Repository erstellen
2. Diese Dateien hochladen
3. Render.com öffnen
4. New + → Web Service
5. GitHub-Repository auswählen
6. Einstellungen:
   - Build Command: npm install
   - Start Command: npm start
7. Deploy klicken
